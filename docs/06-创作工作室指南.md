# 06 - OrangeStudio 创作工作室指南

> AI 音乐创作工作站 · v0.1 设计（v0.6 上线）

## 一、概述

OrangeStudio 是 OrangeRadio 的核心差异化能力：一个**专业级 AI 音乐创作工作站**，由 **MiniMax** 驱动，从灵感到成品全流程 AI 赋能。

```
灵感 ──► AI写词 ──► AI作曲 ──► AI演唱 ──► 多轨编辑 ──► 混音母带 ──► 发布
```

## 二、MiniMax 集成

### Provider 抽象
创作 AI 通过 `AudioAIProvider` trait 抽象，当前主力实现为 MiniMax，但可替换为 Suno/Udio 等：

```rust
#[async_trait]
pub trait AudioAIProvider: Send + Sync {
    fn name(&self) -> &str;
    fn capabilities(&self) -> ProviderCapabilities;
    async fn generate(&self, request: &GenerationRequest) -> Result<GenerationResult>;
    async fn query(&self, task_id: &str) -> Result<GenerationResult>;
}
```

### MiniMax 能力映射
| OrangeStudio 能力 | MiniMax 服务 |
|---|---|
| AI 写词 | MiniMax LLM（对话模型） |
| 作曲编曲 | MiniMax 音乐生成 |
| STEM 分轨 | MiniMax 分轨 + Demucs |
| AI 演唱 | MiniMax TTS / 语音克隆 |
| 创作意图理解 | MiniMax LLM |

### 配置
在设置中填入 MiniMax API Key。代码位于 `crates/orange-studio/src/minimax.rs`。

## 三、全流程详解

### Step 1: 灵感输入
支持三种输入：
- **自然语言描述**："一首 80 年代复古合成器流行，关于夏夜海边，欢快中带点怀旧"
- **哼唱**：录制旋律轮廓，AI 识别并扩展
- **参考音频**：上传参考曲，AI 模仿风格

### Step 2: AI 写词（`LyricsGenerator`）
```rust
let draft = lyrics_gen.generate(&LyricsRequest {
    theme: "夏夜海边".into(),
    mood: "欢快怀旧".into(),
    style: "合成器流行".into(),
    language: "中文".into(),
    structure: vec![Verse, Chorus, Verse, Chorus, Bridge, Chorus],
}).await?;
```
输出结构化歌词（主歌/副歌/Bridge），支持押韵方案控制。

### Step 3: AI 作曲编曲（`Composer`）
```rust
let composition = composer.compose(&CompositionStyle {
    genre: "synthwave".into(),
    tempo_bpm: Some(110.0),
    instruments: vec!["synth", "drum_machine", "bass"].into(),
    mood: "nostalgic upbeat".into(),
}, Some(&draft)).await?;
```
返回完整混音 + STEM 分轨。

### Step 4: STEM 分轨（`StemSeparator`）
生成的曲目自动拆为 5 条独立轨道：
| 轨道 | 用途 |
|---|---|
| 🎤 Vocals 人声 | 可替换为 AI 演唱 |
| 🥁 Drums 鼓组 | 调整节奏 |
| 🎸 Bass 贝斯 | 调整低频 |
| 🎹 Harmony 和声 | 调整配器 |
| 🎵 Other 其他 | 氛围/效果 |

### Step 5: AI 演唱（`VocalSynth`）
- **预设音色**：从音色库选择
- **音色克隆**：上传 30 秒参考音频克隆你的声音
```rust
let voice = vocal_synth.clone_voice("reference.wav", "我的声音").await?;
let vocal_url = vocal_synth.sing(&voice, &lyrics, Some("melody_hint")).await?;
```

### Step 6: 多轨 DAW 编辑
前端 DAW 编辑器（`features/studio/`）：
- **时间线**：拖拽 clip、剪切、对齐
- **钢琴卷帘**：MIDI 编辑（v0.7）
- **混音台**：每轨 EQ/压缩/混响/延迟、音量、声相
- **自动化曲线**：参数随时间变化

### Step 7: 混音母带（`ProjectRenderer`）
```rust
renderer.render(&project, &RenderOptions {
    format: RenderFormat::Wav,
    sample_rate: 48000,
    bit_depth: 24,
    master: true,
    target_lufs: -14.0,
}, "output.wav").await?;
```

### Step 8: 发布
- 加入本地库 → 立即可播放
- 发布到社区创意市场 → 他人可收听/Remix
- 导出音频文件（WAV/FLAC/MP3）

## 四、工程文件格式（.orp）

`StudioProject` 序列化为 JSON，扩展名 `.orp`：
```json
{
  "id": "uuid",
  "name": "夏夜回忆",
  "sample_rate": 48000,
  "bpm": 110.0,
  "musical_key": "A minor",
  "tracks": [
    {
      "name": "人声",
      "kind": "vocals",
      "volume": 0.8,
      "clips": [{ "source": "url", "start_secs": 0, "duration_secs": 180 }],
      "effects": [{ "kind": "reverb", "wet": 0.3, "decay": 2.0 }]
    }
  ],
  "lyrics": "..."
}
```

## 五、创作与播放联动

- 创作完成的曲目**直接进入本地库与播放队列**
- 「懂你模式」学习用户的**创作偏好**，辅助推荐风格匹配的参考曲
- 可发布"原创歌单"，他人能"一起听"创作者的创作过程直播

## 六、版权与署名

- AI 生成内容自动标注「AI 辅助创作」
- 记录创作者署名
- 商用授权管理（后端存证）
- Remix 需遵循原作者授权条款
