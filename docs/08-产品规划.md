# 08 - OrangeRadio 产品规划

> 全面超越 MineRadio · 最后更新 2026-07-01

## 一、产品定位

> **OrangeRadio — 你的智能音乐宇宙**
>
> 一个由 AI 驱动、跨平台、沉浸式、可扩展的下一代音乐伴侣。
> 它不仅是播放器，更是理解你情绪、适应你场景、美化你桌面的智能音乐空间，
> 并内置由 MiniMax 驱动的专业级 AI 音乐创作工作站（OrangeStudio）。

## 四、OrangeRadio 八大超越维度

```
┌─────────────────────────────────────────────────────────────┐
│ 1. 平台覆盖（×5）    2. 技术架构（更轻更快）                  │
│ 3. AI 智能（全方位）  4. 音频引擎（Hi-Res 发烧级）            │
│ 5. 视觉系统（可编程）  6. 社交协作（听歌房间）                │
│ 7. 生态扩展（插件+主题） 8. 场景伴侣（专注+睡眠+创作）         │
└─────────────────────────────────────────────────────────────┘
```

### 维度 1：平台覆盖（×5 于 MineRadio）
- Windows / macOS / Linux / Android / iOS / Web
- 基于 Tauri 2，一套 Rust 核心覆盖全平台

### 维度 2：技术架构（更轻更快）
| 指标 | MineRadio | OrangeRadio |
|------|-----------|-------------|
| 框架 | Electron | Tauri 2 + Rust |
| 安装包 | ~100MB+ | ~10MB |
| 内存 | 高（Chromium 多进程） | 低（原生 Rust） |
| 安全 | 未签名被拦截 | 代码签名 + 自动更新 |
| 启动 | 慢 | 快（原生编译） |

### 维度 3：AI 智能引擎（最大差异化）
- **懂你模式**：AI 行为画像驱动，结合情绪/时段/跳过/收藏动态推荐
- **AI 歌词译注**：实时翻译 + 典故/创作背景标注
- **AI DJ 混音**：BPM 对齐、无缝接歌
- **语音交互**：自然语言点歌、语音控制
- **天气/场景电台**：升级版——天气 + 时间 + 日历 + 情绪多维匹配

### 维度 4：音频引擎（Hi-Res 发烧级）
- FLAC / WAV / ALAC / DSD / APE 无损解码（symphonia）
- 10 段参数 EQ / 空间音频（环绕/双耳）/ 响度归一（EBU R128）
- 母带级品质（24bit/192kHz+ / DSD）

### 维度 5：视觉系统（可编程 Shader）
- Three.js + WebGL 实时渲染：粒子/频谱/流体/3D 歌单架
- **可编程 Shader 场景**（超越 MineRadio 的固定视觉）
- 桌面歌词（多样式）+ 悬浮窗 + 专辑封面动态调色
- 智能光效联动（Hue/RGB 随音乐跳动）

### 维度 6：社交协作
- **一起听**：异地同步播放 + 实时语音/弹幕
- **协作歌单**：多人实时编辑（CRDT）
- **听歌报告/年度回顾**：听歌画像 + 可分享
- **音乐故事/评论社区**

### 维度 7：生态扩展（插件 + 主题工坊）
- **音源插件 API**：实现 `AudioSource` trait 即可接入任意平台
- **可编程视觉场景**：社区创作 Shader 场景
- **主题工坊/创意市场**：皮肤/歌单/视觉场景发布与下载
- **浏览器识歌扩展**：网页 BGM 一键加入库

### 维度 8：场景伴侣 + 独有创作能力
- **智能场景感知**：时间/天气/日历自动切歌单
- **睡眠定时渐弱** + **专注模式/白噪音**
- **投屏与多设备接力**（DLNA/AirPlay）
- ⭐ **OrangeStudio（MineRadio 完全没有）**：MiniMax 驱动的 AI 音乐创作工作站
  - AI 写词 → AI 作曲编曲 → STEM 分轨 → AI 演唱 → DAW 多轨编辑 → 混音母带 → 发布
