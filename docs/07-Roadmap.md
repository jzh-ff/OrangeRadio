## 已知限制（v0.2）
- 频谱为模拟数据（CORS 限制无法用 MediaElementSource 取真实频谱，v0.3 用 Rust FFT 解决）
- 本地库每次启动重新扫描（v0.3 加 SQLite 持久化）
- Windows 用 GNU 工具链（WinLibs），正式发布前评估切 MSVC
- 第三方音源合规性需持续评估
