## 三、播放本地音乐（v0.2 已可用 ✅）

### 第一步：扫描音乐库
1. 点击右上角「**扫描本地音乐**」按钮
2. 在弹出的文件夹选择框中，选择你的音乐文件夹（如 `E:\CloudMusic`）
3. 等待扫描完成，歌曲会显示在列表中

### 第二步：播放
- **双击**列表中的任意歌曲即可播放
- 或鼠标悬停时点击行首的 **▶** 图标

### 支持的格式
FLAC / WAV / ALAC / MP3 / AAC / OGG（无损格式自动标记「无损」徽章）

### 播放控制（底部播放栏）
| 控件 | 功能 |
|------|------|
| ⏮ | 上一首 |
| ⏯ | 播放 / 暂停 |
| ⏭ | 下一首 |
| 进度条 | 拖动跳转（悬停显示滑块） |
| 🔊 音量条 | 拖动调节音量 |
| 模式按钮 | 循环切换：顺序→列表循环→单曲循环→随机→懂你模式 |

### 搜索
在搜索框输入关键词（歌曲名/艺术家/专辑），回车搜索本地库。
