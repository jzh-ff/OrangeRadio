# 02 - OrangeRadio 使用手册

> 版本：v0.3 · 最后更新：2026-07-05

## 一、快速开始

### 启动

```bash
# Windows（需 Rust + Node.js）
./run.ps1
```

首次启动会自动创建 `~/.orangeradio/` 数据目录（库、封面缓存、日志）。

### 界面总览

```
┌────────┬──────────────────────────────────────┐
│        │  全局搜索框（搜索全网音乐）           │
│ 侧边栏  ├──────────────────────────────────────┤
│        │                                      │
│ 搜索   │                                      │
```

## 二、核心功能

### 1. 本地音乐库

- **扫描**：侧边栏「我的音乐库」→「扫描本地音乐」→ 选择文件夹
- 支持 FLAC/WAV/ALAC/MP3/AAC/OGG
- 自动提取标题/艺术家/专辑/封面/时长
- 本地封面提取到 `~/.orangeradio/covers/`

### 2. 全局聚合搜索

- 侧边栏顶部搜索框输入歌名 → 回车
- 并发搜索：本地库 + 网易云 + QQ 音乐 + Spotify + 电台 + 歌曲宝
- 结果混合展示，每行彩色标签标来源（本地/NE/QQ/SP/LIVE/GQB）

### 3. 第三方音源登录

#### 网易云音乐
- Cookie 登录：浏览器登录 music.163.com → 复制 MUSIC_U cookie → 粘贴
- 扫码登录：网易云 APP 扫码

#### QQ 音乐
- Cookie 登录：浏览器登录 y.qq.com → 复制 uin + qqmusic_key cookie
- 扫码登录：QQ APP 扫码（四步换证自动完成）

#### Spotify
- 需配置 Client ID + Client Secret（Spotify Developer 后台申请）
- 30 秒试听（完整播放需 Premium）

#### 歌曲宝（免登录）
- 第三方聚合音源，曲库完整、更新迅速，无需登录或 API Key
- 侧边栏「音源」→「歌曲宝」直接搜索播放
- 已纳入全局聚合搜索（来源标签 GQB）
- ⚠ 实际音频来自酷我等平台，仅供学习研究，商业用途请获取正版授权

### 4. 歌单与收藏

#### 本地歌单系统
- 侧边栏「我的歌单」区 → 点 + 新建歌单
- 任意歌曲行点 **＋** → 弹窗选择歌单
- **支持跨源**：网易云/QQ 歌曲可加入本地歌单，播放时自动取流

#### 收藏
- 歌曲行 / 底部控制栏点 **❤**
- 网易云歌曲 → 收藏到网易云「我喜欢的音乐」远端歌单
- 本地歌曲 → 本地收藏

### 5. 全屏沉浸播放页

点击底部播放栏左侧的**封面**进入全屏页。4 种布局：

| 布局 | 说明 |
|------|------|
| 🎬 电影粒子 | 全屏 Three.js 节拍粒子 + 中央歌词 |
| 🖼 沉浸双栏 | 左封面 + 右歌词滚动 |
| 📜 歌词流 | 全屏居中超大歌词 |
| ▦ 三栏 | 歌词 + 封面 + 评论 |

- ESC 或 ✕ 退出
- 歌词自动跟随播放进度滚动高亮
- 网易云/QQ 歌曲支持翻译歌词

### 6. 热门评论

- 全屏页三栏模式右侧 / cinema 模式点 💬 按钮
- 网易云 + QQ 音乐热门评论（头像/昵称/内容/点赞数）

### 7. 播放队列

- 底部控制栏点 **☰** → 右侧滑出队列面板
- 显示当前播放队列，点击切换播放

### 8. 节奏粒子可视化（cinema 模式）

- 进入 cinema 模式 → 全屏粒子随音乐节奏爆发
- 右上角齿轮（视觉控制台）调节：
  - 节拍灵敏度 / 粒子数量 / Bloom 强度
  - 颜色主题（橙焰/电紫/深海/极光）
  - 镜头晃动开关
- 参数持久化到 localStorage

## 三、播放控制

| 操作 | 方式 |
|------|------|
| 播放/暂停 | 底栏 ▶/⏸ 或双击歌曲 |
| 上一首/下一首 | 底栏 ⏮/⏭ |
| 进度跳转 | 拖动进度条 |
| 音量 | 底栏右侧滑条 |
| 播放模式 | 底栏切换（顺序/列表循环/单曲循环/随机/懂你） |
| 播放队列 | 底栏 ☰ |
| 收藏 | 底栏 ❤ 或歌曲行爱心 |

## 四、数据目录

```
~/.orangeradio/
├── library.sqlite    # 本地库 + 歌单 + 收藏（SQLite）
├── covers/           # 本地音乐封面缓存（FNV 哈希命名）
├── cache/            # QQ 音乐流缓存
├── logs/             # 日志（按天滚动）
└── auth/             # 登录态（AES-256-GCM 加密）
```

## 五、常见问题

### QQ 音乐播放没有声音？
QQ 音乐的 vkey 有有效期（约数小时），如果长时间不操作后无法播放，重新搜索或切歌即可获取新 vkey。

### 网易云歌曲播放失败？
可能需要 VIP 权限，或 cookie 过期。侧边栏重新登录即可。

### Spotify 只有 30 秒？
Spotify 免费版只提供 30 秒试听片段，完整播放需要 Premium 账号 + Playback SDK。

### 本地音乐没有封面？
OrangeRadio 会从音频文件内嵌提取封面。如果文件本身没有内嵌封面，会显示橙色占位。
